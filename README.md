🚀 Deploying on GitHub Pages

Create a repository named Gnanaswarup.github.io
Upload all 4 files: index.html, style.css, script.js, logo.png
Go to Settings → Pages
Under Source, select main branch and / (root)
Click Save — your site will be live at:

   https://gnanaswarup.github.io
